# SiameseChange

Siamese change detection code for pair-wise medical image comparison (a manuscript detailing this approach is submitted)

**Requirements**: 
- Python 3.6.7
- PyTorch 1.1.0
- CUDA 10.0
- GPU support

**How to Run**:

Run scripts in an interactive shell, like IPython.

The main scripts are as follows:

main.py
eval.py
